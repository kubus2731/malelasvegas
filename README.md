# Małe Las Vegas
Projekt z Architektury Komputerowej wykorzystujący mikrokontroler firmy STM
<br /><br />
Q: Na czym jest uruchamiany projekt? <br />
A: Na mikrokontrolerze STMicroelectronics STM32 Nucleo-L476RG
<br /><br />
Q: Jakie dodatkowe perfyferia są wykorzystywane w projekcie?<br />
A: Wyświetlacz OLED oparty działający na szynie SPI1
<br /><br />
Q: W jakim języku jest pisany program?<br />
A: W języku C
